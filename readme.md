## 下载项目文件夹

### 第一种方法

zip下载

### 第二种方法（推荐）

使用git工具

https://git-scm.com/

下载完成后右击桌面（或其他地方），点击Git Bash Here，会蹦出一个对话框，输入

`git clone https://github.com/xrr-233/PKU-Control.git`

即可。
